#  Java OOP Exercises

A collection of Java programming exercises focused on Object-Oriented Programming (OOP) principles and fundamental software design concepts.

This repository was created as part of academic training in Java programming, with emphasis on understanding and applying core OOP principles in structured applications.

---

##  Concepts Covered

- Classes and Objects
- Encapsulation
- Constructors and Method Design
- Basic Object-Oriented Architecture
- Data modeling using Java classes

---

##  Purpose of This Repository

The goal of this repository is to strengthen understanding of:

- Object-Oriented Programming principles
- Code organization and modular design
- Reusability and maintainability of code
- Structuring Java applications using classes and objects

---

##  Technologies Used

- Java (JDK 9+)
- Basic OOP concepts
- Console-based applications

---

##  Project Structure

Each file represents an independent Java exercise focusing on a specific OOP concept or problem.

---

##  Learning Outcomes

- Improved understanding of object-oriented design
- Ability to structure code using classes and objects
- Development of logical and modular programming thinking
